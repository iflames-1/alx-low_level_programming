# Functions, nested loops

### [0-putchar.c](./0-putchar.c)
> Program that prints _putchar, followed by a new line using _putchar.
```bash
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 _putchar.c 0-putchar.c -o 0-putchar && ./0-putchar
```
